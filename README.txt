READ ME file: 
The gitlab repository is composed of various folders.

-exercises : here you will find 5 subfolders composed of all the folders indicating the work from week 1 to week 4 and a final project folder which contains the final project code. Each week ? folder contains some exercises, demos and the weekly project;

-lib : this folder contains subfolders for each library that I used to complete the projects and the exercises;

-src : this folder contains the code the you would like to run in the Arduino board. So for example if you want to run the project of week 2, you go to the folder "week2",copy the code contained inside of the file "project.c" and paste it in the main.c file.
